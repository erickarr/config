Install Pathogen mod:

mkdir -p ~/.vim/autoload ~/.vim/bundle && \
curl -LSso ~/.vim/autoload/pathogen.vim https://tpo.pe/pathogen.vim

To install modules using Pathogen:

cd ~/.vim/bundle && \
git clone git://github.com/altercation/vim-colors-solarized.git

To give gnome-terminal the solarized color scheme:

git clone https://github.com/Anthony25/gnome-terminal-colors-solarized

Run the install script. You can also set the colorized scheme from within Gnome Terminal profile preferences.

Use provided ~/.bashrc for setting up 256 color mode export TERM=xterm-256color
